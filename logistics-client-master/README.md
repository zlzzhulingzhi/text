## 切换至官方源

```
npm config set registry https://registry.npmjs.org
```

## 安装项目依赖

```
npm install
```

### 运行开发环境

```
npm run serve
```

### 打包命令

1. 生产环境
   ```
   npm run build-prod
   ```
2. 测试环境
   ```
   npm run build-test
   ```
3. 压力测试环境
   ```
   npm run build-stress
   ```
